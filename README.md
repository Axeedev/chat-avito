AI-ассистент для повседневных задач с чат-интерфейсом, построенный на базе GigaChat API.
Приложение упрощает поиск информации, позволяет вести диалог с нейросетью и управлять историей общения.

🚀 Возможности

💬 Чат с AI-ассистентом (GigaChat)

🔐 Авторизация через Firebase Authentication

📚 Хранение истории сообщений

🗂 Управление чатами

🔎 Поиск по чатам

👤 Профиль пользователя

🌗 Поддержка светлой и тёмной темы

⚡ Пагинация списка чатов

🔁 Повторная отправка сообщений при ошибке

🏗 Технологический стек


Kotlin

Jetpack Compose

Multi-Module Clean Architecture

MVI (Model-View-Intent)

Nav Component 3 для навигации

Hilt (Dependency Injection)

Firebase Authentication

Room (локальное хранение)

DataStore (настройки приложения)

Coroutines + Flow

Retrofit (работа с API)
